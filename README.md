### How to Run
```
$ bundle
$ be shotgun
```
Visit http://localhost:9393 in your browser and fill out the form!

For example in the test case provided:

> A Level 10 Warrior using an Axe with the following stats:
> 50-65 DMG, 20 STR, 10% Fire Damage
